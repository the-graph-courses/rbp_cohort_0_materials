2 Basic histograms with geom_histogram()

Now let’s use {ggplot2} to plot the distribution of height.

# Simple histogram showing the distribution of height_cm
ggplot(data =  malidd , 
       mapping = aes(x = height_cm)) +
  geom_histogram()
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

It’s hard to get a sense for which range of heights is spanned by each bin in the previous plot, since everything is one big amorphous blob. So let’s add borders around the bins:

# Set border color to "white"
ggplot(data =  malidd , 
       mapping = aes(x = height_cm)) +
  geom_histogram(color = "white")
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

We now have an easier time associating bin ranges to each of the bars.

3 Comparing distributions

We can compare multiple distributions on the same axes.

For example, mapping a categorical variable to the fill aesthetic will create stacked bars.

# Map fill to sex
ggplot(data =  malidd , 
       mapping = aes(x = height_cm,
                     fill = sex)) +
  geom_histogram()
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

3.1 Position adjustments

You can change the position adjustment within geom_histogram().

Possible values for the argument position in geom_histogram() are “stack”, “identity”, and “dodge”.

The default value is “stack”, so adding position = "stack" will give the same plot.

# stack: stacked distributions of a categorical variable  
ggplot(data =  malidd , 
       mapping = aes(x = height_cm, 
                     fill = sex,
                     color = sex)) +
  geom_histogram(position = "stack") + 
  labs(title = "position = 'stack'")
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

Setting position = 'identity' causes the bars to overlap. To see that overlapping, we need to add make the bars slightly transparent by decreasing opacity (alpha).

# identity: draw overlapping distributions on the same plot
ggplot(data =  malidd , 
       mapping = aes(x = height_cm, 
                     fill = sex,
                     color = sex)) +
  geom_histogram(alpha = 0.4, 
                 position = "identity") + 
  labs(title = "position = identity")
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

Setting position = "dodge" places overlapping bars directly beside one another. This makes it easier to compare values.

# dodge: interweave distributions on the same plot
ggplot(data =  malidd , 
       mapping = aes(x = height_cm, 
                     fill = sex,
                     color = sex)) +
  geom_histogram(alpha = 0.4, 
                 position = "dodge") + 
  labs(title = "position = 'dodge'")
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

3.2 Faceting plots

Another way to add additional variables is to split your plot into facets.

When we facet a graphs, it gets split up into multiple panels, one for each level of a categorical variable.

Faceting creates small multiples of a plot, where each panel contains a subset of the data.

# Base plot
ggplot(data = malidd, 
       mapping = aes(x= viral_load))+
    geom_histogram(color="white")
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

3.2.1 Facet wrap

To facet your plot by a single variable, use facet_wrap(). The first argument of facet_wrap() should be a formula, which you create with ~ followed by a variable name. The variable that you pass to facet_wrap() should be discrete.

# Facet by fever
ggplot(data = malidd, 
       mapping = aes(x= viral_load))+
    geom_histogram(color="white")+
    facet_wrap(~fever)
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

Facets are labelled by the difference values of your categorical variable, (e.g., 1 and 0) and not by the variable name itself (e.g., fever).

Facet labels can be modified using the labeller argument. Setting it to label_both will give both the variable name and the value.

# Add more informative panel labels
ggplot(data = malidd, 
       mapping = aes(x= viral_load)) +
    geom_histogram(color="white") +
    facet_wrap(~fever, 
               labeller = label_both)
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

3.2.2 Facet grid

To facet your plot on the combination of two variables, we use facet_grid(). This time the formula should contain two variable names separated by a ~.

# Facet by fever and vomiting
ggplot(data = malidd, 
       mapping = aes(x= viral_load)) +
    geom_histogram(color="white") +
    facet_grid(fever ~ vomit, 
               labeller = label_both)
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.
